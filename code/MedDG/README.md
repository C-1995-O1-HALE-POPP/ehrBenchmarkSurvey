## Overview

MedDG is a large-scale Chinese medical dialogue generation dataset designed for building medical dialogue systems. It contains doctor-patient conversations annotated with entity information and clinical knowledge.

## Task Description

MedDG supports:
- Medical dialogue generation (patient query to doctor response)
- Entity recognition and linking in medical dialogues
- Clinical information extraction from conversations

## Dataset Statistics

The dataset contains over 17,000 dialogues with:
- 12 types of diseases
- Entity annotations for symptoms, drugs, examinations, and diseases
- Multiple dialogue turns per conversation
