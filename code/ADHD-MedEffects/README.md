# UPMC Clinical Benchmarks (7)

These benchmarks are from clinical notes at the University of Pittsburgh Medical Center. All require IRB and data use agreement.

Access: Contact UPMC / respective paper authors.

## Benchmarks

- **ADHD-Behavior** — Detect parent training in behavior management recommendation
- **ADHD-MedEffects** — Detect side effects inquiry in ADHD clinical notes
- **BMT-Status** — Bone marrow transplant status verification
- **HospiceReferral** — Hospice referral eligibility classification
- **ClinicReferral** — Sequoia clinic referral classification
- **ENT-Referral** — ENT referral classification
- **CDI-QA** — Clinical documentation integrity verification QA
- **CLEAR** — Condition history classification from clinical notes
