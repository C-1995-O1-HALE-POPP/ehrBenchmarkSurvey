# Mexican Clinical Records (PPTS)

## Overview
Mexican Clinical Records (PPTS) is a dataset of Mexican Spanish clinical records annotated for named entities and clinical concepts. It focuses on clinical NLP for the Mexican healthcare context.

## Task
- **NER**: Named Entity Recognition on Mexican clinical text
- **Language**: Mexican Spanish
- **Domain**: Clinical records / electronic health records
