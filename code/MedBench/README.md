# MedBench

- **Paper**: MedBench: A Large-Scale Chinese Benchmark for Evaluating Medical LLMs (2023)
- **Type**: Benchmark Suite
- **Status**: ✅ Sub-benchmarks covered

## Sub-benchmarks

Multi-task Chinese medical benchmark covering:
- A1/A2/B Single-Statement Multiple Choice
- A3/A4 Case-Linked Multiple Choice
- Case Analysis, Examinations, Diagnoses, Treatments

## Note

Individual task data was not downloaded separately. The benchmark spans Chinese medical exam-style questions similar to `CNMLE` and `Resident Training` exams.
