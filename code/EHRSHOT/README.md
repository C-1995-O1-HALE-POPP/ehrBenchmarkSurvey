# EHRSHOT

- **Paper**: EHR-R1 (2025), MedAgentGym (2025)
- **Data**: Stanford Medicine EHR (public portion: ~7,000 patients)
- **Access**: Public portion available; full dataset Stanford gated
- **Priority**: P1

## Tasks (15 risk prediction tasks)

**Binary outcomes:**
- Length of Stay (>7d), 30-day Readmission, ICU Transfer
- Anemia, Hyperkalemia, Hyponatremia, Hypoglycemia, Thrombocytopenia
- Acute MI, Celiac, Hyperlipidemia, Hypertension, Lupus, Pancreatic Cancer

**Few-shot & longitudinal:**
- Few-shot EHR Outcome Prediction
- Future Clinical Event Prediction from Longitudinal EHR
