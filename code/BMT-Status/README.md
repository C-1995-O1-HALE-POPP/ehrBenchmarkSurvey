# UPMC Clinical Benchmarks - Quick Reference

This covers 8 IRB-gated clinical benchmarks from University of Pittsburgh Medical Center notes:

| Benchmark | Task | Status |
|---|---|---|
| BMT-Status | Bone Marrow Transplant Status Verification | P3 📋 |
| CLEAR | Condition History Classification | P3 📋 |
| ClinicalNotes-UPMC | Clinical Note Classification | P3 📋 |
| HospiceReferral | Hospice Referral Eligibility | P3 📋 |
| ClinicReferral | Clinic Referral Classification | P3 📋 |
| ENT-Referral | ENT Referral Classification | P3 📋 |
| CDI-QA | Clinical Documentation Integrity QA | P3 📋 |
| BrainMRI-AIS | Brain MRI AIS Assessment (Korean hosp) | P3 📋 |

All require IRB approval and data use agreement.
