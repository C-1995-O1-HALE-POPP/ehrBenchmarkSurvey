# mtsamples (document classification)

- **Paper**: BRIDGE (2025)
- **Data**: MTSamples public medical transcription collection
- **Access**: Public (mtsamples.com, HF, Kaggle)
- **Priority**: P2

## Task

Document/specialty classification from medical transcript text.

Note: "MTSamples" in the benchmark taxonomy refers to document classification, distinct from "MTSamples Procedures" which is treatment plan generation.

- `harishnair04/mtsamples` on HuggingFace (4,999 rows)
