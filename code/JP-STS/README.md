# JP-STS (Japanese Case Reports - Semantic Textual Similarity)

## Overview
JP-STS is a Japanese clinical Semantic Textual Similarity (STS) benchmark based on Japanese clinical case reports. It is part of the NTCIR MedNLP shared tasks.

## Task
- **Semantic Textual Similarity (STS)**
- **Language**: Japanese
- **Domain**: Clinical case reports

## Source
- Part of NTCIR MedNLP shared tasks
- Related to the Japanese case report corpus (MedTxt-CR) and iCorpus
- MedTxt-CR: https://sociocom.naist.jp/medtxt-en/cr/

## Download Status
Dataset not found on HuggingFace or GitHub. This dataset is available through the NTCIR MedNLP organizers and may require direct contact with the NAIST Social Computing Lab.
